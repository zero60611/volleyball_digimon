## 📱 2. 行動端與跨平台手動測試清單 (Cross-Platform Manual Verification)

### A. 選單畫面測試 (Menu Navigation)
- [ ] **按鍵操作**：在起始選單中，點選 `▲` 與 `▼` 虛擬鍵，選單游標必須正常上下切換。
- [ ] **Player 1 確認開局**：點擊 `Z` 鍵確認「1人對戰」，確認進入遊戲後，您控制的是 **左側大耳獸**。
- [ ] **Player 2 確認開局**：點擊 `⏎`（Enter）鍵確認「1人對戰」，確認進入遊戲後，您控制的是 **右側大耳獸**。

### B. 遊戲對戰控制測試 (In-Game Controls)
- [ ] **左右移動**：按住 `◀` 與 `▶` 鍵，您所控制的大耳獸（無論在左在右）必須能滑順左右移動，且放開後大耳獸必須立刻停止。
- [ ] **跳躍動作**：按下 `▲` 鍵，大耳獸必須跳起，且空中可以進行左右微調位移。
- [ ] **向下俯衝**：在大耳獸跳起至空中時，按 `▼` 鍵，大耳獸必須立即向下快速落回地面（快速降落機制）。
- [ ] **擊球/扣殺**：球靠近大耳獸時，按下 `Z` 鍵，必須成功將球擊出並發出擊球特效與聲音。
- [ ] **飛撲防守**：球距離較遠時，同時按 `◀` 或 `▶` 與 `Z` 鍵，大耳獸必須發動斜向飛撲。

### C. UI 與音訊測試 (UI & Audio)
- [ ] **音量切換**：在下拉選項中，切換 BGM 與 SFX 的開/關，確認背景音樂和特效音效能正確靜音或播放。
- [ ] **暫停功能**：在遊戲進行中按鍵盤 `P` 鍵或點選選單「暫停」，確認遊戲畫面與物理運算全部靜止。
- [ ] **自適應畫面**：旋轉手機螢幕或調整瀏覽器視窗大小，確認遊戲 Canvas 畫面有等比例縮放且未發生版面破碎。
